# HR Analytics Dashboard | Power BI

## Overview
Comprehensive HR Analytics Dashboard to analyze employee 
attrition across 1,470 employees with department-level filters.

## Key Metrics
- Total Employees: 1,470
- Attrition Count: 237
- Attrition Rate: 16.1%
- Average Salary: 6.50K
- Average Age: 37
- Average Work Years: 7.01

## Features
- Attrition by Education
- Attrition by Age Group
- Attrition by Salary Slab
- Attrition by Job Role
- Attrition by Years at Company
- Attrition by Gender
- Department-wise filters (HR / R&D / Sales)

## Tools Used
- Power BI Desktop
- DAX
- Power Query (M-Query)
- Excel
